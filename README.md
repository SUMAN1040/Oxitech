## 🌟 Overview

**Oxitech** is an IoT-powered Android application that connects with an external health device to provide **real-time body monitoring**, **SOS emergency assistance**, and **natural air support**. Built with Kotlin and Bluetooth integration, it’s perfect for high-altitude regions and medical environments where immediate health tracking is crucial.

---

## 📱 Core Features

✨ **Live Vital Monitoring**
- ✅ Heart Rate
- 🌡️ Body Temperature
- 🏃 Physical Activity (Walk, Run, Climb)
- 🔥 Calorie Consumption

🆘 **Emergency Support System**
- 🔔 SOS Alerts with One Tap
- 📞 Emergency Contact Integration
- 👨‍⚕️ Auto-connect with Local Doctors (Future Scope)

🔗 **Bluetooth Sync**
- 🔄 Real-time data from Oxitech IoT device
- 📡 Low-latency communication

📊 **User Dashboard**
- 💾 Data Storage and History
- 👤 Account Management
- 🔐 Secure Login/Signup Flow
